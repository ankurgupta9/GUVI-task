# Income Expense Calculator

A simple Income & Expense Calculator built using HTML, Tailwind CSS, and JavaScript.
It supports CRUD operations and saves data using localStorage.

## Features
- Add, edit, and delete income & expense entries
- Filter entries (All / Income / Expense)
- Shows total income, total expense, and balance
- Reset form inputs
- Data persists after refresh
- Responsive design

## Tech Stack
- HTML
- Tailwind CSS
- JavaScript (DOM & localStorage)
